# Yoga App !

# Requirements

*  Java 11+
* MySQL

# Configure MySQL

* create a database in MYSQL nammed `oc_p05_yoga`
* update username and password in [application.properties](./src/main/resources/application.properties)

# Install maven packages

```
mvn clean install
```

# Run dev server

```
mvn spring-boot:run
```

# Run test

```
mvn clean test
```
